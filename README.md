# OMDb Media Finder 🎬

**OMDb Media Finder** is a fast, responsive web application that allows users to search for their favorite movies, TV shows, and games. Built with Vanilla JavaScript and styled using Tailwind CSS, it fetches real-time data from the OMDb API. 

The project features a fully responsive grid layout, dynamic DOM manipulation, and graceful error handling for broken images to ensure a clean user experience across all devices.

## ✨ Features
* **Search Functionality:** Instantly search for any movie or TV show title using the OMDb API.
* **Responsive Design:** A fully responsive grid that adjusts perfectly from mobile screens to wide desktop monitors.
* **Graceful Error Handling:** Automatically detects broken poster links from the API and replaces them with a clean placeholder image.
* **Modern UI:** Styled with Tailwind CSS for a sleek, dark-mode aesthetic.

## 🚀 Tech Stack
* **Frontend:** HTML5, Vanilla JavaScript
* **Styling:** Tailwind CSS
* **Data Fetching:** Axios
* **API:** OMDb API REST endpoints
* **Build Tool:** Vite

## ⚙️ How to Run Locally

If you want to download and run this project on your own machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation

1. **Clone the repository:**
   Open your terminal and run:
   ```bash
   git clone [https://github.com/Tanmayyee/omdb-media-finder.git](https://github.com/Tanmayyee/omdb-media-finder.git)

2. **Go to the project directory:**   
   cd OMDb-movie-finder

3. **Install the dependencies:**
   npm install

4. **Start the development server:**   
   npm run dev
